## 连接远程服务器，并安装v2ray
```
bash <(curl -s -L https://git.io/v2ray.sh)
```

## 获取v2ray url协议连接
```
v2ray url
```

## 安装windows客户端
v2rayN 是Windows平台上一款基于v2ray核心的简洁好用、功能强大的v2ray客户端，支持Vmess、Shadowsocks、Socks5等多种协议，也支持服务器订阅
